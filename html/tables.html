Tabelas

E Pra Quê Tabelas?

Com certeza você já deve ter visto ou até mesmo manipulado uam tabela na vida! Minha primeira experiência com tabelas foi quando brincava com minhas irmãs de Salada de Frutas: era um brincadeira em que escolhíamos vários elementos e depois de sortearmos uma letra do alfabeto, tínhamos que escrever o nome de um desses elementos que começasse com uma dessas letras, quem mais pensasse em elementos diferentes dos seus adversários: podiam ser animais, frutas, nome de atores, programas de tv, cores, obtinha mais pontos. Outras pessoas tem seu primeiro encontro às vezes conflituoso com as tabelas através do Microsoft Excel, organizando informações de estoque, por exemplo, e esse é o conceito que precisamos de tabela: A tabela é uma estrutura de dados construída através de linhas e colunas.

E é claro que nas páginas web teremos uma forma particular de criá-las, é um conhecimento básico e indispensável para um programador web, e relativamente fácil até!

Estrutura

Conseguimos contruir tabelas nas páginas html utilizando as tags <table></table> que conterão a estrutura interna da tabela, as tags <tr></tr> (tr de "table row" - linha da tabela) irão inserir cada uma das linhas, sendo que as tgs <th></th>(th de "table head" - cabeçalho da tabela) utilizaremos para criar a primeira linha da tabela, que corresponde geralmente aos títulos das colunas, isso consequente lhes dará um destaque, e por fim, os elementos das linhas serão inseridos pelas tags <td></td> que envolvem o conteúdo da linha. Veja um exemplo de tabela:

Quadro de Alunos

Tags Semâticas

As tabelas também podem ser escritas utilizando suas tags semânticas própias:

  • A tag de cabeçalho que vamos incluir os títulos das colunas: <thead></thead>;
  • A tag que conterá o conteúdo da tabela: <tbody></tbody>;
  • E por fim a tag de rodapé da tabela <tfoot></tfoot>;

O código não altera em nada o desenho da tabela, poderia ter escrito sem utilizar essas tags, contudo, fiz propositalmmente para anunciar um tópico muito importante que veremos no futuro: as tags semânticas! No mundo dos programadores, onde por diversas vezes um código precisa ser revisto, consertado, alterado, etc. essas tags têm como finalidade facilitar a legibilidade do código entre os propensos editores.

Outra observação: embora tenha utilizado o atributo border para definir a espessura das linhas nesses exemplos, através do CSS podemos realizar toda a estilização das tabelas de maneira mais prática e funcional.
