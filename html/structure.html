A Estrutura Básica de Uma Página Web

O html é parte do tripé que compões aplicações ou páginas web (html, javascript e CSS), não se trata necessariamente de uma linguagem de programação, a sigla significa "Liguagem de Marcação de Hipertexto". Podemos apelidar a estrutura html de "esqueleto" da página web, seu objetivo é "organizar" o conteúdo escrito através de tags que darão forma visual aos dados ao mesmo tempo em que dará sustentação ao conteúdo e acesso os outros elementos, por exemplo: os arquivos javscript e CSS.

Como pode ser visto na imagem que serviu de exemplo anteriormente, o html é construído com "códigos" chamados de "tags" (essa expressão vai se tornar recorrente se você pretende trabalhar com desenvolvimento web), essas tags são comuns a todas as páginas html, elas vão se repetir adquirindo funções específicas conforme a necessidade, pois irão interagir com os navegadores de forma a organizar textos digitados, exibir imagens, vincular formulários, executar scripts, um universo de itens pode ser associados de maneira genérica, as tags serão o ponto central desse assunto, pois para iniciar uma ação teremos de envocar uma tag.

Anatomia do html

Observe no início da página a tag: <!DOCTYPE html> Fig. 1 - linha 1, é o início do documento e indica ser da versão a ser utilizada pelo browser: html5, a mais recente. Pode parecer difícil de observar a primeira vista, mas utilize as linhas numeradas para enxergar o começo propriamente dito na mesma imagem de exemplo linha 2 <html> a abertura e o fechamentona linha 12 </html>, é o conteúdo propriamente dito da página, vamos dividí-lo em duas partes e falar brevemente sobre suas funções: header e footer.

Head

O head pode ser entendido como "cabeçalho" da página, começa pela tag <head> Fig. 1 - linha 3 e termina na linha 8 </head>, contém diversos elementos não expresos diretamente no texto a ser exibido, trabalham por debaixo dos panos por assim dizer. Exemplos: os metadados são expressos através das tags <meta> que transferem informações da página aos navegadores e mecanismos de busca, o título da página < title>Título da Página</title > e também links para para scripts (Js) e folhas de estilo (CSS).

Body

O body é literalmente traduzido do inglês como "o corpo" da html, contido entre as tags: <body> e </body> na Fig.1 linha 9 e na linha 10, que terão o texto a ser lido, imagens, tabelas, vídeos etc. É interessante notar que o que aparece no browser quando estamos vendo uma página web não exibe diretamente a estrutura que digitamos no arquivo html, parte do conteúdo é oculto, isso porque muitas tags tem funções para separar o texto, ou aplicar uma estilização dispensando nesse caso o uso do CSS.
