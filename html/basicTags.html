Tags

A estrutra básica das tags é dada pela junção dos símbolos: "maior" + conteúdo + "barra" + "menor", ficando da seguinte maneira: < conteudo />, algumas tags abrem na esquerda e fecham na direita, outras abrem e fecham na mesma tag. Vejamos algumas tags essenciais:

Tag de Título "h"

A tag <h1></h1> serve para criarmos títulos na página, começa do maior h"1" e vai reduzindo o tamanho conforme variarmos o número para maior, indo até o h"6", veja os exemplos a seguir:

  • <h1>Título</h1>

  • <h2>Título</h2>

  • <h3>Título</h3>

  • <h4>Título</h4>

  • <h5>Título</h5>
  • <h6>Título</h6>

Tags"Strong" ou "b" para Negrito

Tanto a tag <strong>/<strong> como a tag <b>/<b> são usadas para dar um realce ao texto, deixando-o mais "forte". Não há nenhuma diferença entre as duas no resultado:

  • Exemplo usando a tag strong: Negrito.

Ambas as tags <i></i> e <em></em> executam papel semelhante colocando o texto em itálico, que é um realce que "inclina" o texto levemente para a direita, criando um destaque mais "sutil".

  • Exemplo usando a tag i: Itálico.
  • Exemplo usando a tag em: Itálico.
