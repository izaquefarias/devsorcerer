O Que São os Eventos?

Através dos eventos, iremos executar o código javascript na página html, eles são acionados mediante a ações específicas do usuário, exemplo: ao mover o mouse por cima de um texto, ou clicar em um botão de pesquisa, etc.

DOM

Domain Object Model - Modelo de Objeto de Documento: é um modelo da página html, que é referenciada no js como "documento html", o DOM tem um papel intermediário entre o Js e o html: os manipulando o conteúdo das páginas: os elementos, tags, atributos, criando uma "árvore de objetos. No final das contas o DOM é um "modelo de representação de objetos", as aplicações conseguem alterar os dados de um site dispensando a realização de uma atualização manualmente, o próprio usuário pode customizar as páginas navegando por sua estrutura.
